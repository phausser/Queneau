# Queneau Textübersetzer

Textübersetzer nach Queneaus "Stilübungen"

## Voraussetzungen

- Java 17+
- Gradle 7+
- (Optional) Docker

## Build & Start

```bash
./gradlew build
./gradlew bootRun
```

## Tests ausführen

```bash
./gradlew test
```
